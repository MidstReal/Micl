Requires g++ and NASM. 
To install on Ubuntu/Debian:
```bash
sudo apt update
sudo apt install g++ nasm
```
If you have changed the compiler, compile it with the command:
```bash
g++ -o compiler compiler.cpp
```
Then compile the .micl file:
```bash
bash ./compile.sh
```
